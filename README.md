# palerainDev Hub

palerainDev Hub is a game development community where developers can share projects, exchange knowledge, and collaborate on ideas.  
It provides a space for learning, showcasing games, and growing skills for both beginners and professionals.

## Website
https://dev.palerain.com

## Contributing (Feedback & Issues)
This repository is used for:
- Bug reports
- Feature requests
- UI/UX feedback
- Suggestions and improvements

### How to contribute
1. Go to the **Issues** tab
2. Click **New Issue**
3. Choose a template (Bug / Feature Request)
4. Provide as much detail as possible (screenshots help a lot)

## Community
Feel free to share your projects and devlogs on the platform.
Early creators may receive a **Verified Badge** ⭐
